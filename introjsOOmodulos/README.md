# Orientação a objetos em JAVASCRIPT

```javascript

let pessoa = {
    nome: "bla bla bla"
    idade: 210491204
};
```
Object = qualquer estrutura que esteja dentro de chaves e que possua o padrão atributo: valor;

```javascript
pessoa.endereco = "Rua lalala 100";
console.log(pessoa);
```
